# 🤖 AI Email Automation System

An AI-powered workflow that automates email classification, response generation, and structured data logging using n8n and OpenAI.

---

## 🚀 Overview

This system processes incoming emails and automatically:

- Classifies emails (Complaint, Query, Feedback, Urgent, Spam)
- Assigns priority levels (High, Medium, Low)
- Generates professional AI-based replies
- Logs structured insights into Google Sheets

---

## ⚙️ Workflow Architecture

1. Gmail Trigger — captures incoming emails  
2. OpenAI Processing — analyzes email and returns structured output  
3. Data Transformation — extracts category, priority, summary  
4. Google Sheets — stores structured data  
5. Gmail Send — sends automated reply
